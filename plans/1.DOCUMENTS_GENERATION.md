# Feature description
The goal of this product is to produce legal documents based on templates and export those in different formats or send them via email. Scale is a small local business, only dozen of documents are produced per month.

Formats to be supported:
- PDF
- XLS(X)

Before, business has been producing documents by manually composing Excel sheets. Excel sheets from before have to serve as the base on which templates are created. Templates' creation will be a one-time event, but still, templates have to be editable in a user-friendly manner. Currently, we have some buckets for storing documents, but existing in the code references to template management can be disregarded and changed if there is a better solution available.
Of course, Excel usually assumes some math happening within the document. This case is not an exception. Regular invoice types of operations within the documents have to happen.  
Sometimes some documents require stamp/signature. Those should be possible to add as an image in the template and conditionally use in the document.
Audit is an important part of the document management: who created the document, when, was the document shared/exported, etc.

# Constraints:
- Operations must stay within the free tier of a Cloudflare account.
- Documents must respect the legal requirements of the country (Ukraine).
- Prices/units of the entities must be overridable on a case-by-case basis (per document).
